hexo-theme-blank 文档一 ，这篇文章，介绍 blank 主题的安装以及各种页面的生成。

安装

  • 在hexo根目录
  • 修改站点配置文件 _config.yml ，把主题改为 blank

平滑升级

为了主题的平滑升级, blank 使用了 data files 特性。

推荐把主题默认的配置文件 _config.yml 复制到 Hexo 工作目录下的 source/_data/blank.yml，如果 source/_data 的目录不存在那就创建一个。

注意：如果创建了 blank.yml ，它将会替换主题目录下的 _config.yml ，之后你只需要在 blank 主题目录下 git pull 就能平滑升级主题啦！
当你通过 git pull 升级主题后，请检查 _config.yml 是否有新的配置项，如果有，请把新的配置项复制粘贴在 blank.yml

推荐的 Front-matter 设置

参数 描述 默认值
title 标题 文章的文件名
date 时间 文件建立日期
comments 是否显示评论 true
auto_excerpt 是否自动摘要 true
toc 是否显示目录 true
tags 标签 无
categories 分类 无
description 描述 无
