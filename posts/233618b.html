学习 C++ 第一天之通讯录管理系统（共11P）

教学视频：https://www.bilibili.com/video/BV1et411b73Z

因为已经有了 C 的基础，所以直接跳过前边，从P72一直到83，一遍看老师视频，一遍自己敲一遍

  • 设计通讯录结构体和联系人结构体
    在主函数中 AddressBook abs; // 创建通讯录结构体变量
    abs.size = 0; // 初始通讯录中当前人数
  • 用一系列case来实现通讯录的：添加联系人，显示联系人，删除联系人，查找联系人，修改联系人，清空联系人功能
  • 添加联系人功能addPerson(&abs);
  • 显示联系人功能showPerson(&abs);
  • 删除联系人函数deletePerson(&abs)必须先创建检测联系人是否存在函数isExit(&abs);
  • 修改联系人modifyPerson(&abs)
  • 清空联系人cleanPerson(&abs)
